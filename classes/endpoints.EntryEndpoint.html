          linkExtractor?: LinkExtractor,

      ): EntryEndpoint

      Creates a new entry endpoint.

      Parameters

      • uri: string | URL

        The base URI of the REST API.
        Missing trailing slash will be appended automatically.

      • Optionalserializer: Serializer

        Controls the serialization of entities sent to and received from the server.
        Defaults to serializers!JsonSerializer if undefined.

      • OptionalerrorHandler: ErrorHandler

        Handles errors in HTTP responses.
        Defaults to errors!DefaultErrorHandler if undefined.

      • OptionallinkExtractor: LinkExtractor

        Detects links in HTTP responses.
        Defaults to links!HeaderLinkExtractor and links!HalLinkExtractor combined via links!AggregateLinkExtractor if undefined.

      • OptionalhttpClient: HttpClient

        The HTTP client used to communicate with the REST API.
        Defaults to http!FetchHttpClient if undefined.

      Returns EntryEndpoint
